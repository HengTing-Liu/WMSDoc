# BE 经验教训

## 2026-03-29 基础设置模块接口文档

- 坑：接口文档格式需要反复调整，占用较多时间
- 优化：下次先确认文档模板再开始写
- 复用：统一返回格式 R.java 已标准化，可直接参考

## 2026-03-29 环境搭建

- 坑：sudo 权限问题导致安装 Java/Maven 失败多次
- 优化：先确认 sudo 密码格式再执行安装命令
- 复用：echo "密码" | sudo -S 方式可行

## 2026-03-29 项目摸底任务

- 坑：DDL 脚本分散在多个文件中，统计不完整
- 优化：使用 grep 搜索所有 SQL 文件更高效
- 复用：find + grep 组合适合快速定位代码
