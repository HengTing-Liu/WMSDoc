## 第一章 总则

### 1.1 目的
为规范公司仪器设备OEM（原始设备制造商）业务的全流程管理，明确各环节职责与操作标准，确保产品质量、交付周期和客户满意度，特制定本制度。

### 1.2 适用范围
本制度适用于公司所有OEM仪器设备业务的寻源、产品管理、技术支持及售后服务工作。

### 1.3 术语定义
- **OEM（Original Equipment Manufacturer）**：原始设备制造商，即由供应商生产、我司贴牌销售的业务模式
- **ODM（Original Design Manufacturer）**：原始设计制造商，供应商负责设计+生产
- **MOQ（Minimum Order Quantity）**：最小起订量
- **SKU（Stock Keeping Unit）**：库存保有单位

## 第四章 产品定价与资料管理

### 4.1 OEM产品定价模型

#### 定价公式
```
终端售价 = OEM采购价 × (1 + 毛利率) + 运营成本分摊 + 品牌溢价

或反向计算：
OEM采购价上限 = 目标终端价 × (1 - 目标毛利率) - 运营成本
```

#### 成本结构分析
| 成本项目 | 占比参考 | 说明 |
|---------|---------|------|
| OEM采购成本 | 45-55% | 供应商出厂价（含包装） |
| 物流仓储 | 5-8% | 国内运输、海外海运/空运、仓储费用 |
| 关税保险 | 3-5% | 进口关税、货运保险 |
| 营销推广 | 10-15% | 展会、样机、资料、广告投放 |
| 技术支持 | 5-8% | 售前售后人员成本分摊 |
| 管理费用 | 5-8% | 办公、行政、财务分摊 |
| **目标净利润** | **15-25%** | 根据产品竞争力浮动 |

**必备章节：**
1. **安全须知** — 警告标识、安全操作规范、危险源识别
2. **产品概述** — 适用范围、主要功能、技术规格
3. **安装指南** — 开箱检查、安装步骤、环境要求
4. **操作说明** — 界面介绍、功能操作、参数设置
5. **维护保养** — 日常保养、定期校准、耗材更换
6. **故障排除** — 常见问题、故障代码、解决方法
7. **附录** — 保修条款、联系方式、配件清单

**格式规范：**
- 字体：中文使用思源黑体或微软雅黑，英文使用Arial
- 图文比例：每2页至少1张示意图
- 操作步骤：使用带编号的清单，关键动作加粗标注
- 警告信息：红色背景框 + ⚠️图标

## 第七章 供应商绩效管理

### 7.1 供应商评分卡（季度评估）

| 维度 | 权重 | 评分标准 | 数据来源 |
|-----|-----|---------|---------|
| **产品质量** | 40% | 季度不良率<0.5%得满分，每超0.1%扣5分 | 售后故障统计 |
| **交付表现** | 25% | 准时交付率>95%得满分，每低1%扣2分 | ERP交付记录 |
| **价格竞争力** | 15% | 同品类价格对比、年度降价配合度 | 采购记录 |
| **技术响应** | 10% | 技术支持响应速度、问题解决能力 | 技术反馈表 |
| **售后服务** | 10% | 索赔处理效率、备件供应及时性 | 售后反馈 |

#### 供应商等级划分
| 等级 | 得分 | 管理措施 |
|-----|-----|---------|
| **A级-战略** | ≥90分 | 优先采购、联合推广、长期协议 |
| **B级-合格** | 80-89分 | 正常合作，提出改进要求 |
| **C级-观察** | 70-79分 | 限期整改，减少订单份额 |
| **D级-淘汰** | <70分 | 停止新订单，启动替代寻源 |

### 7.2 供应商年度审核

每年至少进行一次现场审核，重点检查：
- 质量管理体系运行有效性
- 上次审核问题整改闭环
- 新品开发能力和项目进展
- 产能扩张及设备更新情况
